### Prerequisites

In order to run this project you need:

1. Install git on your local computer
> Find git installation details here https://git-scm.com/book/en/v2/Getting-Started-Installing-Git

2. Clone this repository
> Run this command `git clone git@github.com:marvin-nyalik/IncognitoReact.git`

3. `cd` to `IncognitoReact` and run `npm run dev`
